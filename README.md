### Getting Started

This is a boilerplate for flutter mobile application with bloc

- Clone this project:

```bash
git clone https://github.com/tvc12/flutter_template
cd flutter_template
```

- Get packages from **pub** for Flutter app:

```bash
flutter packages get
```

- Run project:

```bash
flutter run
```
